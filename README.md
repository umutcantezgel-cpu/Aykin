# Aykin — Custom 3D Druck

Premium-Webplattform für **Aykin**, ein Unternehmen für individuellen 3D-Druck.

## Tech-Stack

- **Framework:** Next.js 16 (App Router)
- **Sprache:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Backend:** Firebase (Firestore, Auth, Admin SDK)
- **Deployment:** Vercel

## Entwicklung

```bash
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Projektstruktur

```
src/
├── app/          # Next.js App Router (Seiten & Layouts)
├── components/   # UI-Komponenten (atoms, molecules, sections)
├── content/      # SEO-Inhalte & Daten
├── hooks/        # Custom React Hooks
├── lib/          # Hilfsfunktionen & Firebase
├── store/        # Zustand State Management
├── styles/       # Globale CSS-Dateien
└── types/        # TypeScript-Typen
```
